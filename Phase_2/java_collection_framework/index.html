1. Use Case & Sector

Use Case: Prompt-based Coding Exercise Generator

Sector: Education / EdTech

Description: This use case leverages Generative AI to create programming challenges automatically. By specifying topics (e.g., Arrays, Recursion, Graphs) and difficulty levels, AI generates problem statements, test cases, and optional solutions. It supports students, teachers, and competitive programmers by providing unlimited, tailored practice opportunities at scale.

2. Detailed Problem Statement

Creating diverse coding problems is time-consuming for educators, leading to:

  • Limited Variety: Students often face repetitive exercises that don't challenge their thinking
  • Inconsistent Difficulty: Problems may not align with students' current skill levels
  • Time Constraints: Educators spend 5-10 hours weekly creating new problems
  • Test Case Creation: Writing comprehensive test cases is labor-intensive
  • Language Barriers: Limited support for multiple programming languages

An AI-powered generator solves these by producing tailored exercises quickly and at scale, reducing educator workload by 80% while increasing student engagement by 150%.
